# TMUX

## session
### start a session
`tmux new -s name`
### session list
`prefix + s`  
`tmux ls`
### attach to session
`tmux a -t mysession`
### detach
`tmux detach`  
`prefix + d`
### kill session
`tmux kill-session -t mysession`
### rename session
`prefix + $`
## windows
### start a window
`prefix + c`
### windows list
`prefix + w`
### close current window
`prefix + ,`
### rename current window
`prefix + &`
